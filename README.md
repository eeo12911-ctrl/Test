# Git 测试项目 (Mock E-commerce Backend)

这是一个用于测试 Git 上传功能的模拟 Node.js 后端项目。

## 项目结构

- `src/`: 源代码
  - `config/`: 配置文件
  - `controllers/`: 控制器逻辑
  - `models/`: 数据模型
  - `routes/`: 路由定义
  - `utils/`: 工具函数
- `tests/`: 测试文件

## 如何使用

1. 安装依赖: `npm install`
2. 以此项目为基础初始化 git 仓库: `git init`
3. 添加远程仓库: `git remote add origin <your-repo-url>`
4. 提交并推送代码
